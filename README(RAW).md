# KIT302 Project 9 WEb VRshowroom

## Introduction
UTAS KIT302 project 9, Web VRshowroom

## Prerequisites
- **Node.js** >= 14
- **npm** >= 10.8.0
- **Git**
- **Lfs**

## How to run
1. Clone this repo from github
2. open terminal and run: 
- npm install
- git lfs install
- git lfs pull
(This will download and replace the LFS file placeholders with the actual content )
3. terminal: 
- npm run dev
